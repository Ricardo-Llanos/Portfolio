Ruta de Aprendizaje

Sesiones de Clase

Cronograma detallado de contenidos, objetivos pedagógicos y herramientas de desarrollo implementadas a lo largo de las primeras 7 semanas académicas.

S1

Introducción al Entorno de Desarrollo y Configuración del Servidor Local

Instalación de herramientas base, configuración de variables de entorno, ciclo de vida de una petición HTTP del lado del servidor y despliegue del primer script en entorno local.

VS Code XAMPP PHP 8.x
S2

Estructuras de Control, Tipos de Datos Avanzados y Funciones Nativas

Implementación de flujos condicionales lógicos, bucles iterativos optimizados, manipulación avanzada de arreglos estructurados y modularización mediante funciones reutilizables.

Algoritmia Backend PHP Vanilla
S3

Programación Orientada a Objetos (POO) del Lado del Servidor

Modelado de clases, instanciación de objetos, aplicación práctica de encapsulamiento, herencia y polimorfismo para construir lógica de negocio escalable.

POO Paradigma Clases abstractas
S4

Persistencia de Datos: Conectividad y Consultas Relacionales SQL

Modelado físico de bases de datos relacionales, establecimiento de conexiones seguras mediante PDO (PHP Data Objects), sanitización de consultas y prevención de ataques SQL Injection.

MySQL PDO Client PHPMyAdmin
S5

Patrón Arquitectónico de Diseño MVC (Modelo-Vista-Controlador)

Separación estricta de responsabilidades web: desacoplamiento de la interfaz de usuario de la persistencia de datos mediante el enrutamiento de peticiones a controladores lógicos centrales.

MVC Pattern Software Architecture
S6

Manejo de Dependencias Backend con Composer y APIs RESTful Básicas

Gestión e instalación de librerías externas del ecosistema Packagist, estructuración de peticiones HTTP semánticas (GET, POST) y serialización de respuestas nativas en formato JSON.

Composer Package JSON API Postman
S7

Fundamentos, Ruteo e Introducción Práctica al Framework Laravel

Estructura de directorios global, configuración del motor de plantillas Blade, sistema de ruteo web semántico e implementación del mapeador relacional Eloquent ORM.

Laravel 12+ Blade Engine Eloquent
S9

Fundamentos de Arquitectura Backend y Entornos de Aplicación Servidor

Introducción a la arquitectura del servidor, componentes backend y despliegue de aplicaciones web en entornos PHP y JavaServer Pages.

Backend Architecture PHP JSP
S10

Desarrollo de Aplicaciones Backend con Python y Programación Orientada a Objetos

Python para backend: sintaxis, estructuras de datos, control de flujo y modelado OOP con clases, herencia y manejo de excepciones.

Python OOP Backend
S11

Framework Backend Django – Patrón MTV, Vistas y Persistencia

Exploración de Django, su patrón MTV, el manejo de URLs, vistas y plantillas junto al ORM para persistencia de datos.

Django MTV Pattern ORM
S12

Administración en Django, Middleware, Sesiones y Seguridad Backend

Configuración del panel de administración, middleware, sesiones y controles de autenticación para asegurar aplicaciones Django.

Django Admin Middleware Seguridad
S13

Diseño e Implementación de APIs RESTful en el Backend

Diseño de APIs REST, serialización, paginación, seguridad y enrutamiento avanzado para servicios backend distribuidos.

REST API Serialización Django REST
S14

Arquitectura de Microservicios en Django, Containerización y Consumo Distribuido

Arquitectura de microservicios, Docker/Kubernetes y consumo distribuido con servicios seguros y desacoplados.

Microservicios Docker Kubernetes
